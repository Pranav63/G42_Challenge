API Endpoints (Port 8000)

POST

Upload Image & Detect Coins

/api/v1/images/upload
Features
  • YOLOv8 detection with confidence scores
  • Data leakage warning for training images
  • Unique UUID for each coin
  • Bounding box, centroid, radius extraction
Response
{
  "image_id": "uuid-string",
  "filename": "coin_image.jpg",
  "coin_count": 4,
  "coins": [
    {
      "id": "coin-uuid",
      "bounding_box": [x, y, width, height],
      "centroid": [x, y],
      "radius": 45,
      "confidence": 0.92
    }
  ],
  "processing_time_ms": 125.3,
  "message": "Detected 4 coins ⚠️ WARNING: This image was in the training set!"
}
